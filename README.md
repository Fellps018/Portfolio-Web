## 📋 Sobre o Projeto

Este projeto é um portfólio profissional desenvolvido com **React** e **Vite**, criado para apresentar minha trajetória como desenvolvedor, minhas habilidades técnicas e os projetos que desenvolvi.

A proposta do site é oferecer uma experiência moderna, responsiva e intuitiva, combinando design elegante com desempenho otimizado.

---

## 🎯 Objetivos

- Apresentar minha identidade profissional.
- Exibir projetos e experiências relevantes.
- Facilitar o contato com recrutadores e clientes.
- Demonstrar conhecimentos em desenvolvimento Front-End.
- Servir como ambiente para experimentação de novas tecnologias e conceitos de UI/UX.

---

## 🖼️ Visão Geral

A página inicial possui:

### Header/Navegação

- Logo/Avatar pessoal.
- Menu de navegação:
  - Apresentação
  - Projetos
  - Destaque
  - Contato
- Botão de ação para iniciar conversa via WhatsApp.

### Hero Section

Título principal:

> Criando Experiências Digitais Únicas

Subtítulo:

> Desenvolvedor Full Stack & Designer

Descrição profissional destacando:

- Desenvolvimento de interfaces modernas.
- Soluções digitais inovadoras.
- Experiências intuitivas e funcionais.

Botões de ação:

- Ver Projetos
- Contato

### Elementos Visuais

- Tema escuro moderno.
- Gradientes em tons de roxo.
- Ilustrações tecnológicas.
- Layout responsivo.
- Efeitos visuais para melhorar a experiência do usuário.
